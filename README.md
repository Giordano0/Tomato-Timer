# 🍅 Tomato Timer

A simple Pomodoro timer built with Python and Tkinter.

## How it works
The Pomodoro technique breaks work into intervals:
- 🟢 25 minutes of focused work
- 🩷 5 minute short break after each work session
- 🔴 20 minute long break after every 4 work sessions

Checkmarks track how many work sessions you've completed.

## How to run
1. Make sure you have Python installed
2. Clone the repo: git clone https://github.com/Giordano0/Tomato-Timer.git
3. Run the script: python main.py

## Requirements
- Python 3
- Tkinter (included with Python by default)
